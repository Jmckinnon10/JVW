Snow Alert

Serverless snow alert system using Node.js, AWS Lambda, Google Sheets API, Weather API, and Twilio System. Retrieves phone numbers and zip codes from Google Sheets and sends SMS alerts using Twilio if snow is forecasted. Includes a Lambda function using a cron job that wakes up every hour and checks the Weather API for snow. Users can submit their information through an HTTP endpoint in order to subscribe.

Technologies:
  • - Node.js
  • - AWS Lambda
  • - Google Sheets API
  • - Weather API
  • - Twilio
